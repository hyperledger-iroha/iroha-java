# Client

The `client` module contains Kotlin implementation of [Hyperledger Iroha2](https://github.com/hyperledger/iroha/tree/iroha2-dev) library.

## Usage

```kotlin
val iroha2Client = Iroha2Client(PEER_URL, log = true)
```
